# Pasta `data`

Esta pasta mantém os arquivos obrigatórios do lab da DIO e adiciona uma camada avançada em `data/crm/`.

## Arquivos principais

- `perfil_investidor.json`: perfil mock completo para teste da Ada.
- `produtos_financeiros.json`: base de produtos, serviços, cartões, benefícios e guardrails.
- `transacoes.csv`: transações fictícias de perfil alta renda.
- `historico_atendimento.csv`: histórico de atendimento consultivo e omnichannel.

## Camada avançada

A pasta `crm/` contém:

- `clientes_atual_v4.csv`
- `historico_12m_v4.csv`
- `next_best_action_v4.csv`
- `base_crm_preditiva_v4.json`

Todos os dados são fictícios e não devem ser usados como cadastro real.
